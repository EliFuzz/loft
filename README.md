## Requirements:
- Docker

## Usage:
1. Start the project:

```bash
make up
```

2. Navigate to [http://localhost:13000](http://localhost:13000)

## Commands:

- Start the project:

```bash
make up
```

- Stop the project:

```bash
make down
```

- Perform cleaning:

```bash
make clean
```
